Lightweight cache cleaner and performance optimizer for WordPress and WooCommerce.

== Description ==

V7 Cache Cleaner is a lightweight plugin that helps speed up your WordPress site by managing cache effectively.

= Features =

* **Page Cache** - Cache HTML output for faster loading
* **Browser Cache** - Add cache headers for static assets
* **Object Cache** - Clear WordPress transients and object cache
* **OPcache** - Clear PHP OPcache when available
* **Auto-Clear** - Automatic cache clearing on content updates
* **Admin Bar Button** - One-click cache clear with size display
* **WooCommerce Support** - Clear cache on product/order updates
* **Third-Party Support** - Works with W3 Total Cache, WP Super Cache, LiteSpeed, Autoptimize

= Auto-Clear Triggers =

* Post/page save, update, or delete
* Plugin or theme updates
* Theme customizer save
* WooCommerce product updates
* WooCommerce order processing

= Privacy & GDPR =

* No data collection
* No external API calls
* No cookies
* No tracking

== Installation ==

1. Upload to `/wp-content/plugins/v7-cache-cleaner/`
2. Activate via Plugins menu
3. Configure via Settings > V7 Cache Cleaner

== Frequently Asked Questions ==

= How do I clear cache? =

Click the "Clear Cache" button in the admin bar or go to Settings > V7 Cache Cleaner.

= Does it work with other cache plugins? =

Yes, it can trigger cache clearing in W3 Total Cache, WP Super Cache, LiteSpeed Cache, and Autoptimize.

= Does it require WooCommerce? =

No, WooCommerce is optional. The plugin works with any WordPress site.
